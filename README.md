# pip_helper
pip工具（辅助用户进行删除，下载，及查看模块/库）
list文件夹：其中的文件用于查看已安装的模块/库,具体启动方式见文件夹中的README.txt文件。
install文件夹：其中的文件用于安装模块/库,具体启动方式见文件夹中的README.txt文件。
uninstall文件夹：其中的文件用于卸载已安装的模块/库,具体启动方式见文件夹中的README.txt文件。
使用前提：
1.下载python时版本>=3.8.10，且下载目录为C:\Users\someone\AppData\Local\Programs\Python\Python38\python.exe
